CONCEITOS BÁSICOS

O Traço Mágico JS é um editor de imagens simples, destinado ao lazer, indicado para desenhos como uma "Lousa Mágica".

Você pode desenhar com Traço Mágico JS utilizando as setas do teclado e/ou o mouse.
Movimente o cursor utilizando o teclado.

  • Use o botão Reta / pra traçar uma reta entre os dois cursores. Atalho letra [ R ].
  • Use o botão Traço ➜ pra traçar o caminho entre o Ponto Anterio ao Ponto Atual. Atalho Letra [ T ].
    Esta opção traz o Ponto Anterior para junto do Ponto Atual.

    DICA: Movendo o Cursor com o teclado o Ponto Anterior (⊕)  não muda de lugar, você pode fazer várias formas repetidas enquando movimenta as setas do cursor e clica nas ferramentas Formas.
    DICA: Você pode utilizar as letras W.A.S.D. para movimentação como mostra a imagem abaixo:






Os Cursores

O Cursor: ⊗ representa o Ponto Atual e o Cursor: ⊕ representa o Ponto Anterior.

As ferramentas de Formas utilizam um ou dois cursores:

  • A Retas serão traçadas utilizando os cursores Ponto Atual (⊗) e Ponto Anterior (⊕) como início e fim.
  • Os Polígonos serão traçados utilizando o cursor Ponto Atual (⊗) como centro.
  • O Clique do mouse irá mudar a posição dos cursores.
  • No modo Previsão os cursores aparecem desta forma: Ponto Atual (⊗) e Ponto Anterior (⊕).
  • O Cursor Ponto Atual (⊗) também pode ser movido com o teclado.

Os Modos

O Traço Mágico JS  pode ser utilizado de dois modos:

  • Modo Imediato: Neste modo as ferramentas traçam suas formas diretamente no desenho.
    O comando PREVISÃO mostra com um LED verde (◆).
  • Modo Previsão: Neste modo as formas aparecem sobre o desenho e podem ser movidas e alteradas à vontade.

    O botão ✔ Envia a forma para o desenho.
    Uma série de mudanças indica que o Modo Previsão está ligado.
    • O comando PREVISÃO substitui o comando IMEDIATO com um LED vermelho (◆) para chamar atenção.
    • No modo Previsão os cursores aparecem desta forma: Ponto Atual (⊗) e Ponto Anterior (⊕).
    DICA: No modo Previsão as setas do teclado movimental o cursor Ponto Atual (⊗).

A Tela de Desenho

A Tela de Desenho é um espaço fixo de 950 x 500 pixels onde o usuário pode desenhar, inserir imagens e textos, ela é composta de 2 camadas

  • Camada Fundo - Esta camada fica atrás do desenho e pode ser alterada pela unicamente pela ferramenta Fundo.
  • Camada Desenho - Esta camada contem todo o desenho do usuário e pode ser alterada por todas as outras ferramentas.




CONTROLES BÁSICOS

O Traço Mágico JS  apresenta controles, eles podem ser identificados pelo título em cor VERDE.
Cada controles afeta uma ou mais ferramentas.

  • A janela de Seleção de Cores é o primeiro tipo de ferramenta.
  • Os Controles Deslizantes operam de forma semelhante vários os controles, apresentando valores máximos e mínimos e uma grande facilidade de uso.
  • A caixa "Formas Solidas" quando marcada define que as formas da linha superior serão traçadas de forma sólida.


Tabela de Cores

Clicando no Círculo da Cor Selecionada abrirá uma janela onde você poderá escolher entre as cores pré-selecionadas ou usar o misturador para ter acesso a milhões de cores.

Fonte

O Controle Tamanho da Fonte afeta as Ferramentas de Texto: Colar Texto e Tabela de Emojis.
O número que aparece acima do controle deslizante é o tamanho da fonte em pixels.
O tamanho da fonte pode ir de 20 a 120 pixels.
O valor padrão é 40.

Lados/Pontas

O Controle Lados afeta todas as Ferramentas Polígono e Estrela.
O número que aparece abaixo do título é o número de lados ou pontas definido.
O Número padrão é 5 mas pode variar de 3 (triângulo) a 30 (triacontágono).
Utilize o Controle Deslizante para mudar o número de lados/Pontas.

Largura do Traço

O Controle Largura do Traço afeta todas as Ferramentas de Formas. O número que aparece abaixo do título é a Largura do Traço definida.
O valor padrão é 2, podendo variar de 2 a 20. Os objetos Sólidos não apresentam borda, portanto este controle não funciona quando o controle Formas Sólidas está marcado.
Utilize o Controle Deslizante para mudar a largura.

Raio/Tamanho

É possível determinar diretamente o tamanho (Largura e Altura) de todas as formas.
O Controle Raio/Tamanho afeta as ferramentas Forma Círculos, Polígonos Retângulos, Estrelas e Imagens inseridas no desenho, define o raio do objeto.
O número que aparece abaixo do título é a largura total da forma definido em pixels.
O valor padrão é 285, podendo variar de 1 a 950.
Utilize os Controles Deslizante para mudar a Largura e Altura das formas.
ATENÇÃO: Utilizando Espiral ou Formas Complexas o comportamento pode ser diferente.

Preenchimento

É possivel criar formas com os seguintes preenchimentos: [V]azio, [S]ólido, Gradiente [L]inear, Gradiente [R]adial e em alguns casos Multicor.
O Controle Transparência afeta todas as ferramentas de Forma e de Fundo que aceitem o controle Formas Sólidas.
O número que aparece abaixo do título é o percentual de transparência definido.
A Transparência padrão é zero [0] (nenhuma transparência), podendo chegar a 99 onde o objeto é 99% transparente.
Utilize o Controle Deslizante para mudar o percentual de transparência.
Ao Marcar uma das opções de Gradiente abrirá uma janela com os controles deslizantes de controle de Gradiente.

Velocidade do Cursor

O Controle Velocidade define o salto do cursor em pixels quando a movimentação é feita com o teclado.
O número que aparece abaixo do título é a velocidade definida.
O valor padrão 20, podendo variar de 1 a 40.




FERRAMENTAS


As Ferramentas podem ser identificados pelo título em cor LARANJA.

FORMAS


As formas incluem Retas, Retângulos, Círculos, Polígonos e Estrelas.

  • As retas são de dois tipos:
    • Use o botão Reta / pra traçar uma reta entre os dois cursores. Atalho letra [ R ].
    • Use o botão Traço ➜ pra traçar o caminho entre o Ponto Anterio ao Ponto Atual. Atalho Letra [ T ].
      Esta opção traz o Ponto Anterior para junto do Ponto Atual.

      DICA: Movendo o Cursor com o teclado o Ponto Anterior (⊕)  não muda de lugar, você pode fazer várias formas repetidas enquando movimenta as setas do cursor e clica nas ferramentas Formas.
      DICA: Você pode utilizar as letras W.A.S.D. para movimentação como mostrado no início deste manual:
  • As Formas (Elipses, Polígonos. Retângulos, Estrelas e Formas Complexas) serão traçadas utilizando o cursor Ponto Atual (⊗) como centro.



FERRAMENTA DESCRIÇÃO DEPENDENCIA
▯ Traça uma retângulo. Ponto Atual (⊗) + Tamanho
◯ Traça um círculo. Ponto Atual (⊗) + Tamanho/Raio
◊ Traça um polígono. Ponto Atual (⊗) + Tamanho/Raio + Lados/Pontas
✩ Traça uma estrela. Ponto Atual (⊗) + Tamanho/Raio + Lados/Pontas
߷ Traça uma espiral. Ponto Atual (⊗) + Tamanho/Raio + Lados/Pontas
ATENÇÃO: Obtenha formas sólidas mancando a caixa "Formas Solidas".
OBS: Todas estas ferramentas são dependentes dos controles Preenchimento e Largura do Traço.



FERRAMENTA DESCRIÇÃO DEPENDENCIA
/ Traça uma reta. Ponto Atual (⊗) e Ponto Anterior (⊕)
OBS: Esta ferramenta é dependente dos controles Largura do Traço e Cor.


FERRAMENTA DESCRIÇÃO DEPENDENCIA
◉ Traça círculos concêntricos. Ponto Atual (⊗) + Tamanho/Raio
✦ Traça formas complexas. Ponto Atual (⊗) + Tamanho/Raio
OBS: Todas estas ferramentas são dependentes dos controles Preenchimento, Largura do Traço e Multicor.

TEXTO

A ferramenta Texto permite inserir textos de até 150 caracteres na imagem com o tamanho entre 10 e 100 pontos. Permite também inserir Emojis diretamente na Tela.
ATENÇÃO: quando o texto não cabe na tela ele será simplesmente truncado! O número que apare entre colchetes é o tamanho do texto em Pontos.

  • Utilize o botão Teclado (⌨) para digitar o texto que será posteriormente colado.
  • Utilize o botão Lápis (✎) para escrever na Tela de Desenho colar o texto no Ponto Atual (⊗).
  • Utilizando o Botão (❤) para inserir Emojis diretamente no Ponto Atual (⊗).
  • Utilize o Controle deslizante de FONTE para mudar o tamanho do texto.
IMAGEM (abrir e salvar)

A ferramenta Imagem permite inserir uma imagem de até 1Mb (Megabyte) na Tela de Desenho.
Após abrir uma imagem no seu computador você poderá inseri-la na Tela de Desenho várias vezes como um carimbo.
O tamanho da imagem é definido da mesma forma que o retângulo: Ponto Atual (⊗) e o Ponto Anterior (⊕) desta forma você é que define o tamanho e a proporção que a imagem deverá aparecer na Tela de Desenho.

Outra função desta ferramenta é fazer o download da Tela de Desenho para o seu computador no formato PNG.

  • Utilize o botão (⏏) para abrir a imagem no seu computador.

  • Utilize o botão (▣) para colar a imagem da mesma maneira que se usa a ferramenta Retângulo.




FERRAMENTAS DE FUNDO

As Ferramentas de Fundo podem ser identificados pelo título em cor AMARELO.

FUNDO

A ferramenta Fundo permite alterar o fundo da Tela de Desenho independentemente. Utilize uma imagem, uma cor ou desenhos aleatórios de círculos e retângulos coloridos. É possível combinar todas e ainda combinar com o controle Transparência. A ferramenta Fundo permite utilizar uma imagem com transparência.
ATENÇÃO: Uma imagem de fundo poderá parecer distorcida a proporção total da Tela de Desenhoo é 950 x 500 pixels.

  • Utilize o botão Cor para definir o fundo com a cor selecionada.
  • Utilize o botão Círculos (⠪) para traçar círculos coloridos aleatórios no fundo.
    DICA: Aumentando o controle Largura do Traço a quantidade de círculos diminui.
  • Utilize o botão Retângulos (⧉) para traçar Retângulos coloridos aleatórios no fundo.
    DICA: Aumentando o controle Largura do Traço a quantidade de retângulos diminui.
  • Utilize o botão Foto para definir o fundo com uma imagem previamente selecionada na ferramenta Imagem.
    DICA: Esta é uma opção para editar uma imagem baixada do Traço Mágico JS.
  • DICA: As 3 últimas opções aceitam a combinação com a ferramenta Transparência.




COMANDOS


Os Comandos podem ser identificados pelo título em cor AZUL.

MODO PREVISÃO

O Modo PREVISÃO ligado permite experimentar todas as Ferramentas (Título Laranja) e os controles que as afetam (Título Verde) antes de aplicar a forma efetivamente ao desenho.

  • Modo Previsão: Neste modo as formas aparecem sobre o desenho e podem ser movidas e alteradas à vontade.

    O botão ✔ Envia a forma para o desenho.
    Uma série de mudanças indica que o Modo Previsão está ligado.
    • Ao ligaro o Modo PREVISÃO um LED vermelho (◆) acende para chamar atenção.
    • No modo Previsão os cursores aparecem desta forma: Ponto Atual (⊗) e Ponto Anterior (⊕).
    • DICA: No modo Previsão as setas do teclado movimental o cursor Ponto Atual (⊗).
    Quando estiver satisfeito com o resultado o botão ✔  envia a forma para o desenho.

    DICA: Aplique a forma em vários locais do desenho apenas clicando em um novo local e utilizando o botão ✔.

    DICA: Use as setas para ajustar a posição da forma durante a previsão.

Com o Modo PREVISÃO desligado você pode desenhar diretamente sobre a tela de desenho.





GRAVAÇÃO

A ferramenta Gravação grava os passos do desenho permitindo que o usuário apague passos a qualquer momento. Todos os passos são gravados, desde o posicionamento do cursor na tela até as mudanças de fundo, uma maneira prática de dizer isso é: todos os passos das ferramentas à esquerda da ferramenta Gravação são gravados.

ATENÇÃO: Não é possível gravar os passos no computador, apenas fazer o download da imagem.

Quando um passo é apagado todas as ferramentas retornam para a configuração existente no momento da gravação do passo anterior.
No título da ferramenta vemos o LED de gravação que indica a situação da gravação: Gravação Ligada: ✹ ou Gravação Desligada: ✹.
Logo abaixo temos o contador de passos gravados: Passo[1].

  • Utilize o botão Pausar (❚❚) para pausar a Gravação.
  • Utilize o botão Play (►) Refazer todo o desenho gravado, mesmo depois de um Apaga.
    ATENÇÃO: Depois de usar o botão Play (►) a gravação retorna desligada..
  • Utilize o botão Apaga (⌫) para apagar o último passo.
    DICA: Não há como refazer o passo que foi apagado. / Só é possível apagar os passos sequencialmente!
  • Utilize o botão (✘) para apagar toda a gravação.
    ATENÇÃO: Depois disso não será mais possível apagar passos no desenho.
SALVAR, APAGAR E REINICIAR

A ferramenta Apagar permite apagar a tela e a ferramenta Reset reinicia todo o programa.

  • SALVANDO SUA TELA DE DESENHO EM DISCO:
    Utilize o botão SALVAR para fazer o download da sua Tela de Desenho para seu computador.
  • Utilize o botão APAGA para apagar toda a Tela de Desenho e torna-la branca.
    DICA: Você pode utilizar o botão Play (►) para refazer o desenho caso haja uma gravação na memória.
  • Utilize o botão RESET para reiniciar todo o programa.
    ATENÇÃO: Se usar a função RESET antes de salvar (SALVAR) o desenho no computador nada poderá ser recuperado!
